Welcome to GDXR Studiosdocumentation!
===================================

**GDXR** (Game Dev XR) is a studio developing XR Content.
You can find out more by visiting <https://gdxrstudio.co.uk/>`_
and offers a *simple* and *intuitive* API.

how to :ref:`installation` the project.
